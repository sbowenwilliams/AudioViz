Intro

This is the boilerplate that you see on the right. We will use that as a base to how to use the rendering loop to make animations. Each world got a rendering loop. You can hook function to this loop.

Playground 
<!doctype html><title>Minimal tQuery Page</title>
<script src="../../build/tquery-all.js"></script>
<body><script>
    var world = tQuery.createWorld().boilerplate().start();
    var object = tQuery.createTorus().addTo(world);
</script></body>

Hook the rendering loop

Now that we got a basic tQuery page. We will hook function in the rendering loop. This function will be executed at every iteration of this loop. So everytime the 3D scene is rendered.

Playground 
<!doctype html><title>Rendering loop and Animation</title>
<script src="../../build/tquery-all.js"></script>
<body><script>
    var world = tQuery.createWorld().boilerplate().start();
    var object = tQuery.createTorus().addTo(world);
    world.loop().hook(function(){
        /* ... your animation code goes here */
    });
</script></body>

Make the object move

Inside the hooked function, we rotate the object by 0.01 radian. We do that at every iteration of the loop, so the object is moving.

Playground 
<!doctype html><title></title>
<script src="../../build/tquery-all.js"></script>
<body><script>
    var world = tQuery.createWorld().boilerplate().start();
    var object = tQuery.createTorus().addTo(world);
    world.loop().hook(function(){
	object.rotateX(0.01);
    });
</script></body>

Done!

Now we animated an object in the rendering loop. This loop is executed during a requestanimationframe for efficiency. It is possible to hook functions at various stage of the rendering. This loop is the base of most animation in tQuery.
