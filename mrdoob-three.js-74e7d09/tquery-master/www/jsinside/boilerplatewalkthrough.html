Intro

This is the boilerplate that you see on the right. This is the base of most of tQuery application. I will walk you thru it. 
<!doctype html><title>Minimal tQuery Page</title>
<script src="tquery-all.js"></script>
<body><script>
var world = tQuery.createWorld().boilerplate().start();
var object = tQuery.createTorus().addTo(world);
</script></body>

First an empty page

Let's start by an empty index.html. The code you see is a valid HTML5 page. 
<!doctype html><title></title>
<body><script>
<!-- Your code goes here -->
</script></body>

Let's include the library

We added a script tag to load the library. Additionnaly we add a title thus it is easier 
<!doctype html><title>tQuery Boilerplate</title>
<script src="tquery-all.js"></script>
<body><script>
</script></body>

Let's create a world

Now we got a page which load tquery. Nice! but still it doesnt do much. We will create a tQuery world to our page. 
<!doctype html><title>tQuery Boilerplate</title>
<script src="tquery-all.js"></script>
<body><script>
    var world = tQuery.createWorld().boilerplate().start();
</script></body>

Now we add an object

Our page is now creating a world. Cool! but still still it isn't too visual. Let's add a object to our world. 
<!doctype html><title>tQuery Boilerplate</title>
<script src="tquery-all.js"></script>
<body><script>
    var world = tQuery.createWorld().boilerplate().start();
    var object = tQuery.createTorus().addTo(world);
</script></body>

Done!

Now we understand tQuery boilerplate. It includes the library. Then create a world, add the boilerplate, and then start it. We just have to add a torus to our world and we are done! Only 6 lines btw.
